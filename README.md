# 🎙️ Voice Tone Sentiment & Speech Emotion Recognition (AI/ML Project)


## 📌 Project Overview

This project focuses on detecting human emotions from speech using Machine Learning and Deep Learning techniques. It analyzes voice tone and classifies emotions such as happy, sad, angry, and neutral.

---


## 🎯 Objective

To build a model that can recognize emotions from audio data and help in applications like:

* Customer sentiment analysis
* Voice assistants
* Mental health monitoring

---


## 📊 Features

* Audio preprocessing (noise removal, feature extraction)
* Feature extraction using MFCC
* Emotion classification model
* Model evaluation and accuracy analysis

---


## 🧠 Technologies Used

* Python
* Machine Learning / Deep Learning
* Librosa (audio processing)
* NumPy, Pandas
* Scikit-learn / TensorFlow

---


## 📁 Project Structure

* data/ → dataset (or sample)
* notebooks/ → Jupyter notebooks
* models/ → trained models
* src/ → source code

---


## 📈 Output

The model predicts emotions such as:

* Happy
* Sad
* Angry
* Neutral

---


## 🚀 How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the notebook or script:

```bash
python main.py
```
